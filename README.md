# Project Overview

### This project contains automation test scripts developed using Selenium WebDriver with Java to test web application functionalities.
The framework is designed to perform automated testing for web elements such as forms, tables, buttons, and navigation flows.
The goal of this project is to improve test efficiency, reduce manual testing effort, and ensure application quality through automated test execution.

## Technologies Used

Java,
Selenium WebDriver,
TestNG,
Maven,
GitHub,
IntelliJ IDEA,

## Features
- Automated web element interaction
- Dynamic waits using WebDriverWait
- XPath and locator strategies
- Form automation
- Web table automation
- Test execution using TestNG
- Reusable automation scripts

## Test Scenarios Covered
- Login page automation
- Form input automation
- Button click actions
- Web table handling
- Element validation
- Navigation and UI interaction
